O 1º passo para iniciar a utilização do IW-Mobile será entrar em contato com a empresa de Home Care para a qual você presta serviços e obter os dados necessários para o cadastramento de um servidor no IW-Mobile.

Solicite as seguintes informações:

  1. Host (IP ou DNS do Servidor)
    (Ex.: iw.iwsoftware.com.br)
  2. Port IP (Número da porta IP)
    (Ex.: 9290)
  3. Profile
  4. Usuário
    Nome do usuário que você usará para acessar o IW da empresa de Home Care.
    Se você já acessa o IW (versão desktop) dessa empresa, o usuário/senha são os mesmos que você já vem utilizando.

Quando você tiver as informações (1), (2), (3) e (4) descritas acima, então, na tela inicial do IW-Mobile toque no botão "Ajustes" e selecione a opção "Servidores". Será aberta uma interface para o cadastramento dos dados do Servidor da empresa de Home Care.

Nessa interface preencha os seguintes campos:

    Nome para esse servidor
    (ex.: Nome da Empresa de Home Care que forneceu os dados acima)
    Host
    Host informado pela empresa - Informação (1).
    Port
    Port informado pela empresa - informação (2).
    Profile
    Port informado pela empresa - Informação (3).
    Usuário
    Seu Usuário informado pela empresa - Informação (4).
    e finalmente toque no botão "salvar"

Notas:

  • Você poderá cadastrar vários servidores
    (ex.: se você trabalha para mais de uma empresa de Home Care, cadastre um servidor para cada uma delas).
  • Após cadastrar os dados do servidor retorne para a tela inicial.
  • Agora o(s) servidor(es) cadastrado(s) já aparecerá(ão) na caixa de seleção de servidores.
  • Para iniciar a utilização do IW-Mobile:
    Selecione o servidor desejado, digite a sua senha de acesso ao IW da respectiva empresa e toque no botão "Login".
  • Recomendações Técnicas:
    Para dispositivos que usam sistema operacional Android, recomendamos uso de aparelhos que possuam mínimo de 1 Gbyte de memória "RAM".
    #IMPORTANTE#:
    Não confundir memŕia RAM com memória de armazenamento.
    Aparelhos atuais , mesmo os mais básicos, já costumam vir equipados com mínimo de 8 GBytes para armazenamento.
    A quantidade de memŕia RAM, não é uma informação que costuma aparecer nos catálogos comerciais. Os vendedores podem não saber dar essa infomação.
    Mas existe um modo facil de obter.
    Tem diversas aplicações gratuitas no Google Play que fornecem todos os dados. Ex: Hardware Info.
